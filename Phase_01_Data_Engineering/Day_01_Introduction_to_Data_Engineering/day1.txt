A hospital stores patient X-ray images, doctor's notes, and a table of patient vital signs. Classify each data type.
Patient X-ray images → Unstructured Data
Doctor's notes → Semi-structured / Unstructured Data
Table of patient vital signs → Structured Data
Explanation:
Structured Data = Data arranged in rows and columns
Semi-structured Data = Partly organized data
Unstructured Data = Images, videos, audio, etc.

Companies need Data Engineers to collect, clean, store, and manage large amounts of data.
They create data pipelines and databases so companies can analyze data easily.

Write Python code to load sales.csv and show its first 10 rows.
Answer:
import pandas as pd
df = pd.read_csv("sales.csv")
print(df.head(10))
Explanation:
read_csv() loads the CSV file
head(10) shows the first 10 rows
What does df.shape return? If it returns (500, 15), what does each number mean?
Answer:
df.shape returns the number of rows and columns in a DataFrame.
If it returns:
(500, 15)
It means:
500 = Number of rows
15 = Number of columns
How do you filter a DataFrame called orders to show only rows where city is Mumbai?
Answer:
orders[orders["city"] == "Mumbai"]
Explanation:
This filters and shows only the rows where the city is Mumbai.
